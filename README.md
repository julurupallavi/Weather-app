# Python Weather App 🌦️

A simple Weather Application built using Python that fetches live weather data from the OpenWeatherMap API.

## Features

- Get real-time weather information
- Displays:
  - City Name
  - Country
  - Temperature
  - Feels Like
  - Humidity
  - Weather Description
  - Wind Speed

## Technologies Used

- Python
- REST API
- Requests Library
- JSON Parsing

## API Used

OpenWeatherMap API

https://openweathermap.org/api

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Python-Weather-App.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Get a free API key from OpenWeatherMap.

4. Replace:

```python
API_KEY = "YOUR_API_KEY"
```

with your own API key.

5. Run

```bash
python weather.py
```
